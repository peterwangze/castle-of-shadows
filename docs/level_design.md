# 关卡设计文档

## 城堡总体布局

### 区域划分
```
暗影城堡地图（非线性探索）
┌─────────────────────────────────────┐
│          城堡外围                   │
│   入口广场 → 吊桥 → 大门            │
├─────────────────────────────────────┤
│          一层：接待区域             │
│   大厅 → 画廊 → 宴会厅 → 厨房       │
├─────────────────────────────────────┤
│          二层：居住区域             │
│   客房 → 书房 → 主卧室 → 阳台       │
├─────────────────────────────────────┤
│          地下：功能区域             │
│   地牢 → 墓穴 → 实验室 → 储藏室     │
├─────────────────────────────────────┤
│          高层：特殊区域             │
│   钟楼 → 天文台 → 塔楼 → 阁楼       │
├─────────────────────────────────────┤
│          核心：最终区域             │
│   王座厅 → 仪式室 → 深渊传送门      │
└─────────────────────────────────────┘
```

### 解锁机制
- **能力门锁**：需要特定能力才能通过
  - 二段跳：高平台
  - 蹬墙跳：垂直通道
  - 暗影潜行：激光栅栏
  - 破防攻击：脆弱墙壁
- **钥匙物品**：击败Boss或完成支线获得
- **时间门**：某些区域只在血月时段开放

## 详细区域设计

### 区域1：城堡外围 (教学区)

#### 入口广场
- **尺寸**：160×96像素（5屏宽度）
- **地形**：碎石路，倒塌雕像，干涸喷泉
- **敌人**：骷髅士兵×3，血蝠群×2
- **教学元素**：
  - 基础移动：左右移动，跳跃台阶
  - 基础战斗：攻击骷髅，格挡教学
  - 环境互动：推动石块触发机关
- **秘密**：喷泉下隐藏通道（需要后期潜水能力）

#### 吊桥区域
- **机制**：破损吊桥，需要找到两个机关启动
- **谜题**：两个机关分别被敌人守卫，需击败后启动
- **敌人**：石像鬼×2（机关守卫），怨灵×1
- **Boss战前准备**：提供第一个血瓶和武器升级

### 区域2：大厅与画廊 (战斗训练区)

#### 主大厅
- **布局**：三层结构，中央大楼梯，两侧回廊
- **敌人配置**：
  - 地面层：骷髅士兵×4，狂化村民×2
  - 回廊层：怨灵×3（远程攻击）
  - 吊灯：血蝠群×2（俯冲攻击）
- **垂直探索**：学习使用二段跳到达高处平台
- **隐藏房间**：壁炉后密室（需要炸开）

#### 艺术画廊
- **特色**：移动的画中幽灵，画框可作为平台
- **谜题**：按正确顺序激活画像（线索在画作标题）
- **敌人**：画中幽灵×6（从画框跳出攻击）
- **奖励**：解锁副武器"圣水瓶"

### 区域3：地下墓穴 (恐怖探索区)

#### 墓穴通道
- **光照**：仅玩家火把照明，视野有限
- **机制**：黑暗恐惧值（长时间黑暗会持续掉血）
- **敌人**：大量骷髅士兵，新敌人"腐尸"（缓慢但高防）
- **环境危害**：
  - 陷阱地板：触发后射出骨刺
  - 毒气区域：持续中毒伤害
  - 深坑：即死掉落（需跳跃通过）

#### 墓室大厅 (Boss战：腐化骑士)
- **竞技场**：圆形大厅，四根柱子可躲避
- **阶段设计**：
  1. 骑士常规攻击，柱子可破坏
  2. 召唤骷髅士兵，需要清理杂兵
  3. 狂暴冲锋，利用柱子躲避路线
- **战后**：解锁"破防攻击"能力，可破坏脆弱墙壁

### 区域4：血色宴会厅 (平台挑战区)

#### 宴会主厅
- **动态环境**：移动的餐桌，摇摆的吊灯
- **平台挑战**：在餐桌和吊灯间跳跃前进
- **敌人**：幽灵侍从×8（端着毒酒盘）
- **特殊机制**：红色地毯区域安全，木地板会塌陷

#### 厨房区域
- **环境危害**：滚烫炉灶，飞溅热油
- **敌人**：厨师生魂×4（投掷厨具）
- **谜题**：调整四个炉灶温度解锁冷库门
- **奖励**：获得"耐热靴"（免疫火焰地形伤害）

### 区域5：钟楼之巅 (垂直挑战区)

#### 钟楼内部
- **垂直设计**：256像素高度，多层平台
- **机制**：摆动钟摆可作为平台，但会随时间摆动
- **敌人**：石像鬼×6（定点守卫），蝙蝠群×4
- **时机挑战**：需要在钟摆正确位置时跳跃

#### 钟楼顶部 (Boss战：血月女巫)
- **圆形平台**：边缘会崩塌，战斗区域逐渐缩小
- **阶段机制**：
  1. 女巫在中央施法，玩家绕圈躲避
  2. 召唤血祭坛，必须摧毁才能伤害Boss
  3. 全屏血雨，需要站在钟摆遮挡下
- **战后**：解锁"空中冲刺"能力

### 区域6：深渊核心 (最终区域)

#### 古代遗迹通道
- **环境**：混合城堡建筑和古老石造遗迹
- **敌人**：所有类型敌人混合出现
- **资源补给**：大量血瓶和弹药，为最终战准备
- **剧情揭示**：散落的日记页揭示背叛者真相

#### 仪式大厅 (最终Boss战：德古拉)
- **多阶段战场**：
  1. 大厅对决：常规剑术战
  2. 蝙蝠形态：追击战，需要射击空中蝙蝠
  3. 真身解放：巨大化Boss，攻击特定弱点
  4. 传送门核心：保护莉亚娜关闭传送门
- **动态环境**：可破坏的窗帘（引入阳光），可启动的圣水装置
- **隐藏结局条件**：在战斗中收集三个圣物可触发完美结局

## 关卡机制细节

### 检查点系统
- **存档点**：魔法蜡烛，触碰保存进度
- **重生规则**：从最近存档点重生，恢复50%生命
- **快速旅行**：后期解锁传送阵网络

### 难度调节
- **敌人强度**：随玩家等级动态调整
- **可选挑战**：限制血瓶使用可获得额外奖励
- **辅助模式**：降低平台难度，增加检查点

### 秘密收集
- **古籍页**：50页，收集齐全解锁真结局
- **灵魂碎片**：隐藏Boss掉落，兑换传奇装备
- **开发者房间**：输入特定密码可进入彩蛋房间

## 地图设计规范

### 瓦片集规格
- **基础瓦片**：16×16像素，4种地形类型
  - 石砖地板（安全）
  - 木板（可能破损）
  - 泥土（减速）
  - 尖刺（伤害）
- **装饰瓦片**：32×32像素，环境装饰
- **机关瓦片**：动画效果，触发区域

### 关卡尺寸标准
- **标准房间**：160×96像素（10×6个屏幕单位）
- **走廊**：32×96像素或160×32像素
- **Boss房间**：240×144像素（1.5倍标准）
- **垂直关卡**：96×256像素

### 导航提示
- **视觉引导**：光源指示前进方向
- **声音提示**：敌人靠近有独特音效
- **地图标记**：发现秘密区域后地图上标记

## 进度节奏

### 前期（1-2小时）
- 学习基础操作
- 击败第一个Boss
- 解锁首个关键能力

### 中期（3-6小时）
- 探索主要区域
- 击败3个Boss
- 解锁大部分能力
- 发现主要剧情转折

### 后期（2-4小时）
- 收集关键物品
- 挑战隐藏Boss
- 最终Boss战
- 结局选择

### 完成度追求（额外2-3小时）
- 100%收集率
- 真结局解锁
- 速通挑战
