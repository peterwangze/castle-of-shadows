# 美术设计规范

## 像素艺术风格指南

### 总体风格
- **分辨率**：320×180像素基础分辨率（HD 16:9）
- **像素比例**：1:1像素，无抗锯齿
- **风格参考**：16-bit时代（SFC/MD）的恶魔城系列，结合现代清晰度
- **颜色深度**：32色调色板（含透明）

### 调色板设计

#### 情景调色板
- **墓穴区域**：增加蓝绿调，减少红色
- **宴会厅**：增加红色和金色

- **最终区域**：增加紫色和黑色，减少饱和度

## 主角像素设计

### 艾登·夜行者 (32×48像素)

#### 站立姿势（正面）
```
帧1：准备姿态，右手握剑
头部：3×3像素椭圆，深棕色头发
身体：16×24像素，深褐色皮甲
腿部：12×16像素，深棕色裤子
披风：8×32像素，暗红色，底部飘动
武器：长剑（12×2像素），银色
```

#### 动画帧规范
1. **跑动动画**：8帧循环
   - 帧1-2：右腿前，身体前倾
   - 帧3-4：双腿并拢，身体最高
   - 帧5-6：左腿前，身体前倾
   - 帧7-8：双腿并拢，身体最低（缓冲）
   - 披风：每帧飘动角度不同

2. **跳跃动画**：3帧
   - 起跳：下蹲压缩（32×40像素）
   - 空中：伸展（32×48像素）
   - 落地：下蹲缓冲（32×40像素）

3. **攻击动画**：4帧
   - 预备：剑后拉（帧1）
   - 挥砍：剑弧光（帧2-3，带残影）
   - 收招：恢复（帧4）

4. **潜行动画**：4帧半透明循环
   - 透明度：50%→70%→50%→30%
   - 颜色：增加蓝色调
   - 特效：淡淡雾状轨迹

### 敌人像素设计

```
骨架：白色（#e6e6e6）带灰色阴影
```

```
颜色：蓝绿色渐变（#00c896→#2d4a78）
核心：中心亮光（#ffffff）
动画：4帧浮动（上下移动+形变）
攻击：发射能量球（8×8像素）
```

## Boss像素设计

### 腐化骑士·雷纳德 (48×64像素)

#### 第一阶段（骑士形态）
```
盔甲：残破板甲，深灰色带血迹
披风：破碎暗红披风
武器：双手巨剑（24×4像素）
细节：头盔裂缝透出红光
动画：沉重步伐，盔甲摩擦声效
```

#### 第二阶段（狂暴形态）
```
变化：舍弃左肩甲，露出黑色骨架
武器：双持（左右手各一把剑）
特效：黑暗能量从裂缝涌出
颜色：增加红色调，眼睛全红
```

#### 第三阶段（亡灵形态）
```
变化：盔甲几乎全部破碎
核心：胸腔内黑暗核心发光
特效：持续掉落骨片粒子
攻击：远程剑气（16×64像素波）
```

## 环境艺术设计

### 瓦片集规格
- **基础瓦片**：16×16像素，4向自动瓦片
- **装饰元素**：32×32像素，手绘细节
- **前景元素**：半透明，增加深度感
- **背景层次**：3层视差滚动

### 区域视觉特色

#### 城堡外围
- **颜色**：冷灰色调，雾气效果
- **细节**：破损石墙，枯树，墓碑
- **天气**：飘落枯叶粒子

#### 地下墓穴
- **照明**：火炬动态光影
- **纹理**：潮湿石壁，青苔斑点
- **特效**：飘浮尘埃粒子

#### 宴会厅
- **色彩**：暖色调，红色金色为主
- **细节**：烛台，挂毯，银器
- **光影**：吊灯摇摆阴影

#### 钟楼
- **垂直感**：强烈透视线条
- **机械**：齿轮，链条，钟摆动画
- **时间**：随时间变化的天空背景

## 用户界面设计

### HUD元素 (屏幕布局)
```
左上角：生命值（心形图标×10）
右上角：暗影能量（蓝色条）
底部中央：武器图标（当前装备）
右下角：副武器数量（数字显示）
```

### 菜单界面
- **风格**：羊皮纸卷轴，墨水笔迹
- **字体**：像素字体，5×7像素大小
- **交互**：选项高亮时轻微放大
- **动画**：翻页效果，墨水晕开

### 地图系统
- **显示**：迷雾探索，已探索区域亮起
- **标记**：自定义图钉，颜色编码
- **缩放**：三级缩放（区域→楼层→全局）

## 特效规范

### 攻击特效
- **剑击**：白色弧光，4帧消失
- **重击**：黄色冲击波，6帧扩散
- **背刺**：红色叉形闪光，8帧
- **远程**：能量球，带拖尾轨迹

### 魔法特效
- **圣水**：蓝色扩散波纹
- **血魔法**：红色粒子螺旋
- **暗影**：黑色烟雾，逐渐消散
- **净化**：金色光柱，向上飘散粒子

### 环境特效
- **火炬**：4帧火焰动画
- **雨滴**：随机下落，地面溅起
- **雾气**：缓慢飘动的半透明层
- **血迹**：随时间淡化的红色污渍

## 导出规范

### 文件格式
- **图像**：PNG（无损），索引颜色模式
- **图集**：2048×2048像素最大尺寸
- **命名**：类型_名称_状态_帧数.png
  - char_player_idle_01.png
  - enemy_skeleton_walk_02.png
  - boss_vampire_attack_03.png

### 目录结构
```
assets/art/
├── characters/
│   ├── player/（玩家所有动画）
│   ├── enemies/（按类型分类）
│   └── bosses/（每个Boss独立文件夹）
├── tilesets/
│   ├── castle/（城堡瓦片）
│   ├── cave/（洞穴瓦片）
│   └── interior/（室内瓦片）
├── ui/
│   ├── hud/（游戏界面）
│   ├── menu/（菜单元素）
│   └── fonts/（像素字体）
└── effects/
    ├── combat/（战斗特效）
    ├── magic/（魔法特效）
    └── environment/（环境特效）
```
