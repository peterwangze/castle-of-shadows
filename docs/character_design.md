# 角色设计文档

## 主角设计

### 艾登·夜行者 (玩家角色)

#### 外观设计
- **身高比例**：6头身像素比例，修长敏捷体型
- **服装风格**：深褐色皮甲猎魔装，暗红披风，银色护肩
- **标志特征**：
  - 左眼下方有猎魔人印记（发光符文）
  - 背部交叉背负主武器和副武器
  - 腰挂各种狩猎工具（银瓶、绳索、小刀）
- **颜色方案**：深棕（皮甲）、暗红（披风）、银灰（金属）、深蓝（内衬）

#### 动作设计
- **移动**：流畅的跑动、二段跳、蹬墙跳
- **战斗**：挥剑有重量感，重击带残影效果
- **特殊**：暗影潜行时身体半透明化，留下雾状轨迹
- **受击**：硬直时间短，受击闪光红色

#### 能力成长
```
初始能力：
- 生命值：100
- 攻击力：15
- 防御力：10
- 敏捷：20
- 暗影能量：50

可解锁技能树：
1. 战斗系：
   - 连续斩击（3连击→5连击）
   - 蓄力重击（破防效果）
   - 旋风斩（范围攻击）

2. 敏捷系：
   - 二段跳→三段跳
   - 空中冲刺
   - 蹬墙跑

3. 暗影系：
   - 潜行持续时间延长
   - 潜行攻击背刺加成
   - 暗影分身（诱饵）
```

## 敌人设计

### 普通敌人分类

#### 亡灵类 (Undead)
- **骷髅士兵**
  - 外观：破碎骨架，锈蚀武器
  - 行为：缓慢移动，近战攻击
  - 弱点：圣水、银武器
  - 掉落：骨片（合成材料）、少量金币

- **怨灵**
  - 外观：半透明幽灵，漂浮移动
  - 行为：穿透墙壁，远程能量弹
  - 弱点：盐圈、圣光
  - 特殊：物理攻击免疫50%

#### 黑暗生物类 (Dark Creatures)
- **血蝠**
  - 外观：红色蝙蝠群，聚合形态
  - 行为：群体攻击，俯冲叮咬
  - 弱点：火把、范围攻击
  - 特殊：击杀一只会分裂成两只小蝙蝠

- **石像鬼**
  - 外观：石质雕像，翅膀可展开
  - 行为：定点守卫，玩家接近激活
  - 弱点：锤类武器、雷电
  - 特殊：远程石化射线（短暂定身）

#### 被侵蚀者 (Corrupted)
- **狂化村民**
  - 外观：衣衫褴褛，眼睛血红
  - 行为：不规则移动，抓挠攻击
  - 弱点：头部攻击、镇定药剂
  - 掉落：村民遗物（任务物品）

- **暗影猎犬**
  - 外观：黑色毛皮，熔岩般裂纹
  - 行为：快速突进，扑咬连击
  - 弱点：腹部、冰属性
  - 特殊：死亡后爆炸释放暗影弹

## Boss设计

### 第一章Boss：墓穴守护者

#### 腐化骑士·雷纳德
- **背景**：前城堡卫队长，被德古拉转化为亡灵骑士
- **外观**：
  - 残破的板甲，裂缝中渗出黑暗能量
  - 双手持巨剑，剑身缠绕锁链
  - 头盔下只有两点红色幽光
- **战斗阶段**：
  1. **骑士姿态**：常规剑术攻击，盾牌格挡
  2. **狂暴姿态**（HP<50%）：舍弃防御，双持攻击
  3. **亡灵姿态**（HP<20%）：召唤骷髅士兵，自身远程释放剑气
- **攻击模式**：
  - 地面三连斩（可格挡）
  - 冲锋突刺（需跳跃躲避）
  - 召唤地刺（区域预警）
- **弱点**：背部护甲裂缝、圣属性攻击
- **掉落**：骑士大剑（武器）、守护者徽章（钥匙物品）

### 第二章Boss：血月女巫

#### 塞蕾娜·血月
- **背景**：德古拉的法师顾问，精通血魔法
- **外观**：
  - 深红长袍，漂浮离地
  - 双手凝聚血球，头发无风自动
  - 面部被魔法面纱遮挡
- **战斗阶段**：
  1. **魔法咏唱**：远程血弹，召唤血蝠
  2. **仪式强化**（HP<60%）：地面出现魔法阵，增强所有法术
  3. **血月降临**（HP<30%）：变为无敌状态，必须破坏三个血祭坛
- **攻击模式**：
  - 追踪血弹（慢速追踪）
  - 血液喷泉（脚下区域）
  - 血蝠风暴（全屏躲避）
- **弱点**：打断咏唱、净化符文
- **掉落**：血魔法书（副武器）、女巫钥匙（区域解锁）

### 最终Boss：吸血鬼领主

#### 德古拉·暗影
- **背景**：暗影城堡主人，千年吸血鬼
- **外观**：
  - 优雅的贵族服饰，深红内衬
  - 苍白的皮肤，尖牙，血红瞳孔
  - 战斗时化为蝙蝠群，可瞬间移动
- **战斗阶段**：
  1. **贵族决斗**：使用细剑，优雅的剑术
  2. **黑暗解放**（HP<70%）：释放吸血鬼真身，爪击和吸血
  3. **蝙蝠形态**（HP<40%）：化为蝙蝠群，多方位攻击
  4. **绝望反击**（HP<10%）：全屏黑暗爆发，需要躲避到光柱中
- **攻击模式**：
  - 瞬移背刺（快速闪现）
  - 血矛投射（直线攻击）
  - 暗影束缚（定身效果）
  - 吸血恢复（接触玩家回血）
- **弱点**：阳光（场景中有可拉开窗帘）、银桩、心脏部位
- **掉落**：暗影之心（剧情物品）、领主披风（顶级装备）

## NPC设计

### 友好NPC

#### 铁匠：布兰登·铁锤
- **外观**：壮硕矮人，满身煤灰，大胡子
- **位置**：城堡外围秘密营地
- **功能**：武器升级、装备修理
- **对话**：粗鲁但热心，讨厌吸血鬼"那些尖牙的娘炮"

#### 商人：神秘旅者
- **外观**：兜帽遮面，总带着神秘微笑
- **位置**：随机出现在安全屋
- **功能**：出售稀有物品、情报交易
- **特殊**：可用特殊货币"灵魂碎片"交易传奇物品

#### 学者：艾尔维拉
- **外观**：戴眼镜的年轻女学者，抱着一堆古籍
- **位置**：城堡图书馆（中期解锁）
- **功能**：翻译古籍、提供谜题提示、解锁新技能
- **支线任务**：收集散落的古籍页

## 美术规范

### 像素规格
- **角色尺寸**：主角32×48像素（站立）
- **动画帧数**：跑动8帧，攻击4帧，跳跃3帧
- **调色板**：主要使用32色调色板，暗色调为主
- **特效**：粒子效果使用8×8像素粒子

### 设计原则
1. **轮廓识别**：每个角色剪影独特，易于辨认
2. **颜色编码**：敌人按类型使用不同主色（亡灵=蓝绿，黑暗生物=深红）
3. **动作反馈**：受击闪光、攻击特效清晰明显
4. **环境互动**：角色与场景光影统一

## 声音设计

### 角色音效
- **艾登**：轻快的脚步声，剑刃破风声，受伤吸气声
- **骷髅**：骨骼摩擦声，锈剑挥舞声
- **怨灵**：空灵哀嚎声，能量凝聚声
- **Boss**：专属主题音乐，阶段转换音效

### 语音提示
- **艾登**：发现秘密"嗯？"，低生命值"需要治疗..."
- **导师**（通讯）："小心身后！"，"尝试绕到侧面"
- **德古拉**："凡人，你很有趣"，"感受永恒的黑暗吧"
